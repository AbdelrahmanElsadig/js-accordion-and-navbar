  • How many other names are there for Javascript?

    +

    JavaScript, was originally named Mocha, later it was renamed to LiveScript, and then to JavaScript. The language was then submitted for standarization to the ECMA International Organization. By that time, Netscape didn't allow the use of the "JavaScript" name, so the standarized language is named ECMAScript.
